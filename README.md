# Bitcoin Price Prediction

This project focuses on predicting Bitcoin prices using historical data and machine learning techniques.  
The goal is to analyze past price trends and build a model that can estimate future prices.

# Project Overview
- Data preprocessing and cleaning
- Exploratory data analysis (EDA)
- Feature selection
- Model training and prediction
- Visualization of results

# Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

# Files in this Repository
- `Bitcoin_price_prediction.ipynb` – Jupyter Notebook containing the complete implementation of the project

# How to Run
1. Open the notebook in **Google Colab** or **Jupyter Notebook**
2. Run all cells sequentially to see the results

# Use Case
This project can be used as a beginner-level machine learning project to understand time-series data and prediction concepts.
